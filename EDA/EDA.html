Mouse Wait classification

This is a study of machine learning and its application to mouse wait time in computers. Specifically, a classification of mouse wait time estimates based on dynamic and static system information within the 2020 time interval. Dynamic system information is subject to GUID, as each system has its own set of configuration, while static system information would mitigate the effect of such bias, as it provides information for each GUID. The model reaches an accuracy of 70% with Decision Tree Classifier.
